Project Extras

Distance calculation and timing functions

As an extra to the main model, timing functions were used to test how long various calculations took as the number of iterations increased. In this instance, the time taken to calculate all the distances between agents was calculated as the number of iterations increased by magnitude 10. After the run times had been calculated, they were plotted against a logarithmic scale of iterations so that the relationship between the two variables could be seen.

Functions to calculate the minimum and maximum distances between agents were also added, ensuring that agents were not being compared against themselves.
